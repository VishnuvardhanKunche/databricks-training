# Week 5 - Phase 1: Referring to Columns in PySpark

---

## What We Did Today

### Referring to DataFrame Columns

- Selected columns using column names as strings
- Filtered rows using string expressions
- Selected columns using DataFrame column notation
- Filtered rows using DataFrame column objects
- Used the `col()` function to reference columns
- Used bracket notation to access DataFrame columns
- Compared multiple approaches for selecting and filtering data

### SQL and PySpark Comparison

- Wrote SQL queries for column selection and filtering
- Implemented the same operations using PySpark DataFrame APIs
- Compared SQL syntax with PySpark syntax to understand their similarities

---

## PySpark Functions Used

| Function | Description |
|----------|-------------|
| `select()` | Selects one or more columns |
| `filter()` | Filters rows based on a condition |
| `col()` | References DataFrame columns |
| `show()` | Displays DataFrame contents |

## Key Concepts Learned

| Concept | Purpose |
|---------|---------|
| Column Selection | Retrieve specific columns from a DataFrame |
| Row Filtering | Extract records based on conditions |
| String Column Reference | Access columns using column names as strings |
| DataFrame Column Notation | Reference columns using DataFrame objects |
| `col()` Function | Explicitly reference DataFrame columns |
| Bracket Notation | Access DataFrame columns using indexing syntax |
| SQL to PySpark | Convert SQL queries into equivalent PySpark operations |
