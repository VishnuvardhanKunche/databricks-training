# Offline Databricks Training - Day 1 (14-07-2026)

---

## Overview

This repository contains the first offline assignment from the Databricks Offline Training Session conducted on **14-07-2026**. The assignment focuses on strengthening fundamental SQL concepts by solving a variety of queries using a sample Employee dataset.

The objective of this assignment is to build a strong SQL foundation before moving to advanced data engineering concepts in Databricks and PySpark.

---

## Topics Covered

The assignment includes practice on the following SQL concepts:

- SELECT statements
- WHERE clause
- GROUP BY
- HAVING
- ORDER BY and LIMIT (Top Records)
- DISTINCT
- Comparison Operators
- Logical Operators
- IN and NOT IN
- BETWEEN
- LIKE Operator

---

## What Was Done

- Created the **Employees** table using DDL statements.
- Inserted sample employee records into the table.
- Solved **55 SQL practice questions** covering different SQL operations.
- Practiced writing optimized SQL queries for filtering, grouping, sorting, and aggregation.
- Improved understanding of SQL syntax and query writing.
